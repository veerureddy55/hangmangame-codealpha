*Hangman Game - CodeAlpha Python Internship
Task 1: Hangman Game
Description
This is a simple text-based Hangman game developed using Python as part of the CodeAlpha Python Programming Internship.

The player guesses a hidden word one letter at a time. The game selects a random word from a predefined list and limits incorrect guesses to 6.

Features
Random word selection
Limited incorrect attempts
Console-based input/output
Simple and beginner-friendly logic
Concepts Used
Python basics
random module
while loop
if-else conditions
strings and lists
How to Run
Type or open hangmangame.py in VS Code and run the code.
After running, save the file on your desktop and drag it into the GitHub repository as it is.
Follow the on-screen instructions to play the game.
Internship
This project is completed as Task 1 for the CodeAlpha Python Programming Internship.
